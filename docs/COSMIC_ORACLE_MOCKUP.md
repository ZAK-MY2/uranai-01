## 🎨 カラーパレット＆アニメーション効果

### ダークモード（メイン）
```
背景: #0a0a1f → #1a1a3e (呼吸する深宇宙グラデーション)
テキスト: #e5e7eb (柔らかい星の光)
アクセント: #9333ea (神秘的な紫)
強調: #fbbf24 (星の輝き)
カード背景: rgba(255, 255, 255, 0.05) (グラスモーフィズム)
```

### ライトモード（切替可能）
```
背景: #fafafa → #f3f4f6 (雲のような柔らかい白)
テキスト: #1f2937 (深い宇宙色)
アクセント: #9333ea (変わらず神秘的)
強調: #f59e0b (太陽の輝き)
カード背景: rgba(0, 0, 0, 0.03) (繊細な影)
```

### アニメーション効果

1. **背景の呼吸** (8秒サイクル)
   - 微かなスケール変化 (1.0 ↔ 1.02)
   - グラデーション色の変化

2. **星のまばたき** (3-12秒ランダム)
   - opacity: 0.3 ↔ 1.0
   - ランダムな位置に配置

3. **カードの浮遊** (20秒サイクル)
   - Y軸: 0px → -8px → -12px → -5px → 0px
   - 微細な回転: 0° → 0.5° → 0° → -0.5° → 0°

4. **ホバー効果** (0.8秒)
   - transform: translateY(-8px) scale(1.02)
   - glow effect: box-shadow拡大

5. **テキスト出現** (1.5秒)
   - opacity: 0 → 1
   - translateY: 30px → 0px  
   - blur: 10px → 0px
